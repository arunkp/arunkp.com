Web Designer & Front-end Developer

Reponsive web design makes a website look great and adaptive to any screen size. Using the tools above mentioned, i create mockups and interfaces for websites and mobile applications.

Every project comes with a problem in hand to solve which is what makes a product. I design with users in mind, so i strongly stress on the point of usablity.

My Design Process involves:

  • Discussions - A clear picture on what a client aims to achieve, any others sites he likes and wants me to follow similar approach, extra help he wants such as hosting, domain name registration, email to be set up
  • Research - Thorough lookup over the design elements required for this particular project
  • Wireframing - Rapid design of Layout and placement of content on the website or application
  • Brand Guidelines - This is optional, if you need style guides for the later use.

I work on both sketch and photoshop.
